# 其他按鈕

## 收藏本頁面的所有作品

在使用者頁面或者搜尋頁面裡，點選下載面板的“其他”選項卡，可以看到這個按鈕。它會自動擷取當前頁面的所有作品，自動收藏，自動新增 tag。

![](./images/20200701173159.png)

**注意：**
- 圖片和小說頁面都可以支援；
- 只擷取**當前**頁面（1 頁），不會擷取後續其他頁面。
- 收藏時直接收藏所有作品，不會根據擷取條件進行過濾。

?>執行過程中，這個按鈕上會顯示進度資訊。

### 幫未分類的作品加入tag

![](./images/20200316104507.png)

>只有在自己的收藏頁面裡才有這個按鈕。

在自己的收藏頁面裡，點選下載面板的“其他”選項卡，可以看到這個按鈕。

如果你自己收藏的作品裡有一些沒有新增 tag，你可以使用這個按鈕，給它們自動新增 tag。

?>執行過程中，這個按鈕上會顯示進度資訊。

## 將使用者頭像另存為圖示檔案

![](./images/2020-10-29_112940.png)

在使用者頁面或者作品頁面裡，點選下載面板的“其他”選項卡，可以看到這個按鈕。你可以把該使用者的頭像儲存為 ico 圖示。

點選之後下載器會把使用者頭像儲存為 256*256 畫素的 ico 檔案。

?>ico 檔案可以設定成資料夾圖示，所以你可以把畫師資料夾的圖示設定成畫師的頭像。
