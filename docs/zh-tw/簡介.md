# 簡介

Pixiv Batch Downloader 是一個 Chrome 瀏覽器擴充套件程式，用於批次下載 [Pixiv.net](https://www.pixiv.net/ ':target=_blank') 上的圖片和小說。

它可以設定篩選條件，設定命名規則，也可以建立資料夾。

支援把動圖轉換為 webm 或者 gif 或者 apng；把小說儲存為 txt 或者 epub 格式。

支援多種頁面類型，支援斷點續傳。

本程式也有一些輔助功能，如去除廣告、快速收藏、看圖模式、給未分類作品新增 tag 等。
