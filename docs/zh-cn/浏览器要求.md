# 浏览器要求

本程序可以运行在 Chromium 内核的浏览器上。

## PC

只要能够安装最新版的 Chrome 浏览器，应该都可以使用这个扩展程序。

推荐使用 Chrome 浏览器。Microsoft 的新版 Edge 浏览器也可以。

!>请始终使用最新版本的浏览器。

!>不能在 Firefox 浏览器上使用。（因为一些兼容性问题）

!>不要使用国产的套壳浏览器。

## Android

在 Android 上你可以使用 Yandex 浏览器，因为它可以安装 Chrome 浏览器的扩展。

> 因为本程序只为 PC 设计，所以在 Android 上的浏览器中使用时，请在浏览器菜单里选择“桌面版本”或类似选项，以 PC 浏览器的模式浏览 Pixiv。

## IOS

!>无法使用

因为 IOS 上的浏览器都必须使用 Safari 浏览器的内核，并且无法安装扩展，所以在 IOS 上无法使用这个扩展。
