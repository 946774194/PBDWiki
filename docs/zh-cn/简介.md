# 简介

Pixiv Batch Downloader 是一个 Chrome 浏览器扩展程序，用于批量下载 [Pixiv.net](https://www.pixiv.net/ ':target=_blank') 上的图片和小说。

它可以设定筛选条件，设置命名规则，也可以建立文件夹。

支持把动图转换为 webm 或者 gif 或者 apng；把小说保存为 txt 或者 epub 格式。

支持多种页面类型，支持断点续传。

本程序也有一些辅助功能，如去除广告、快速收藏、看图模式、给未分类作品添加 tag 等。
