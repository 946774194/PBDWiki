# 尝试初次下载

我们打开一个测试页面，这里是 [画师 まふゆ 的作品列表页面](https://www.pixiv.net/member_illust.php?id=5229572&type=illust ':target=_blank')：

![](./images/2019-07-30-15-32-22.jpg)

点击页面右侧的下载按钮，打开设置面板，然后点击“开始抓取”按钮，本程序就会自动开始抓取作品。

## 日志

本程序运行期间，会在页面顶部显示一些简单的提示，包含设置的条件和下载进度。例如：

```
任务开始
下载所有页面
排除作品类型：动图
多图作品下载前 1 张图片
正在抓取，请等待……
当前有 38 个作品
已获取 38 个图片网址
抓取完毕！

正在下载中
38 / 38

√ 下载完毕!
```

?>第一次进行批量下载时，浏览器可能在地址栏的位置询问：是否允许此网站下载多个文件？选择允许。

![](./images/2019-07-31-10-16-38.png)

## 标题上的状态

本程序进行时，会在网页的标题前面添加一些符号，提示任务的状态。例如：

![](./images/2019-07-30-14-06-03.png)

![](./images/2019-07-30-14-06-53.png)

![](./images/2019-07-30-14-05-40.png)

常用的状态如下：

- ↑ 抓取中
- ↓ 下载中
- √ 下载完毕

还有一些不常用的状态：

- → 抓取完毕，等待下一步操作（tag 搜索页）
- ▶ 抓取完毕，可以开始下载
- ║ 下载暂停
- ■ 下载停止

## 保存位置

下载的文件存放在**浏览器的下载目录**里。
