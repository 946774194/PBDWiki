# 如何更新

本程序会从 Github 上检查新版本。如果有新版本可用，设置面板的右上角会显示蓝色的更新按钮：

![](./images/2019-09-18-16-28-21.png)

?>由于浏览器的应用商店需要审核，所以当提示有新版本时，应用商店里可能还是当前版本。在这种情况下，如果你想要使用最新版本的话，请离线安装新版本。

## 自动更新

从浏览器的扩展商店里安装的扩展程序会自动更新。

## 手动更新

手动更新需要下载新版本的安装包，手动更新。其实就是离线安装。

1. 打开本程序的 [GitHub releases 页面](https://github.com/xuejianxianzun/PixivBatchDownloader/releases ':target=_blank') 。
2. 下载最新版本的第一个 zip 文件，解压得到 powerfulpixivdownloader 文件夹。
3. 把 powerfulpixivdownloader 文件夹放到和旧版本相同的位置，覆盖旧版本的文件。
4. 重启浏览器。

当你重新打开浏览器时，就可以使用新版本了。
